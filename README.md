# StayEase 🏨

StayEase is a modern hotel booking platform that allows users to search, compare, and book accommodations with ease. Built using the MERN stack, StayEase offers an intuitive interface, secure booking features, and real-time hotel availability to enhance the travel experience.

## 🚀 Features

- 🌍 Search hotels by location
- 💰 Filter by price, ratings, and amenities
- 🛎️ View detailed hotel information with images
- 🔒 Secure and simple booking process
- 👤 User authentication and profile management
- 📦 Admin panel for hotel management (add, update, delete listings)

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT
- **Version Control**: Git & GitHub
